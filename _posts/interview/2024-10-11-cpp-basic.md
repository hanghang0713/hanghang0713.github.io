# C++ 中的基础知识

## C++ 中有哪些不同的数据类型
#### 1. 原始数据类型
- int
- char
- wchar_t
- float
- double
- void
...

#### 2. 派生数据类型
- function
- array
- pointer
- reference

#### 3. 用户定义的数据类型
- struct
- class
- enum
- union
- typedef defined 

## C 和 C++ 的区别是什么


| C              | C++                                             |
|----------------|-------------------------------------------------|
| 面向过程       | 面向对象                                        |
| 不支持数据隐藏 | 通过封装隐藏数据，确保数据结构和运算符按预期使用 |
